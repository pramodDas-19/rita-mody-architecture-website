# 🏛️ Rita Mody – Architectural Portfolio Redesign

A premium architectural portfolio redesign for **Rita Mody**, focused on delivering a modern, editorial-style digital experience with luxury UI/UX and smooth animations.

---

## 🚀 Project Overview

This project is a complete redesign of the existing Rita Mody website, transforming it into a **high-end architectural portfolio**.

### 🎯 Goals

* Elevate brand presence with a premium look
* Improve visual hierarchy and readability
* Showcase projects as case studies
* Implement smooth, modern animations
* Maintain original content while upgrading design

---

## 🎨 Design System

### Theme

* Background: `#F8F8F8`
* Cards/Sections: `#FFFFFF`
* Accent: `#C5A46D` (Muted Gold / Champagne)
* Text Primary: `#111111`
* Text Secondary: `#666666`
* Borders: `#E5E5E5`

### Typography

* **Headings:** Cormorant Garamond
* **Body:** Inter

### Design Style

* Minimal
* Editorial
* Luxury architectural feel
* Clean spacing and layout

---

## ⚙️ Tech Stack

* **Next.js** (App Router)
* **React.js**
* **Tailwind CSS**
* **Framer Motion** (animations)
* **GSAP + ScrollTrigger** (advanced motion effects)

---

## 🎬 Animation System

## 📦 Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/rita-mody-portfolio-redesign.git
cd rita-mody-portfolio-redesign
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```
